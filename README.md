# Financial Goal Progress Tracker
追踪财务目标进度，提供缺口分析、优先级排序和定期检查框架。

## What it does
- 🎯 目标可行性分析：根据当前储蓄与收入计算目标缺口
- 📊 多目标优先级排序：按紧急性、重要性和时间范围分配资金
- 📋 定期检查模板：净值快照、进度仪表盘、差异分析与调整建议

## Example scenarios

**1. 两年内存够首付买房**
> 👤 我跟我对象计划两年内存30万首付买房结婚，目前两个人月入共2万左右，每月开销1.2万，能实现吗？
> 🤖 计算目标可行性：每月可存8000元，2年=19.2万，距离30万差10.8万。提供增额方案：涨薪估算、年终奖利用、低风险理财、减少非必要支出。综合评估后给出差距分析和具体行动计划，推荐使用目标储蓄卡强制储蓄。

**2. 计算购房首付缺口**
> 👤 Track my progress toward a $50K house down payment in 3 years. I have $12K saved and save $800/month.
> 🤖 Gap analysis showing $12K current + $28.8K planned = $40.8K (shortfall $9.2K). Three acceleration scenarios with adjusted monthly savings, side-income targets, and quarterly milestone checkpoints.

**3. 多目标优先级排序**
> 👤 I have 5 financial goals competing for the same funds — help me prioritize.
> 🤖 Priority matrix scoring goals on urgency, importance, and time-horizon. Recommended allocation: emergency fund first → high-interest debt → retirement match → down payment.
