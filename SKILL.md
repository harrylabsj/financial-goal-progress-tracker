# Financial Goal Progress Tracker

## Overview

Tracks financial goal progress. This is a descriptive skill that provides frameworks and templates without executing real code.

## Usage Scenarios

### Scenario 1

**User input:** "Track my progress toward a $50K house down payment in 3 years. I have $12K saved and save $800/month."

**Expected output:** Gap analysis showing $12K current + $28.8K planned = $40.8K (shortfall $9.2K). Three acceleration scenarios with adjusted monthly savings, side-income targets, and quarterly milestone checkpoints.

### Scenario 2

**User input:** "I have 5 financial goals competing for the same funds — help me prioritize."

**Expected output:** Priority matrix scoring goals on urgency, importance, and time-horizon, with recommended funding allocation and sequencing (emergency fund first → high-interest debt → retirement match → down payment).

### Scenario 3

**User input:** "Set up a quarterly financial goal review ritual for our household."

**Expected output:** Quarterly review template with net-worth snapshot, goal-progress dashboard, variance analysis, celebration triggers, and next-quarter adjustment recommendations.
### Scenario 4: 两年内存够首付
**User input:** "我跟我对象计划两年内存30万首付买房结婚，目前两个人月入共2万左右，每月开销1.2万，能实现吗？"
**Expected output:** 计算目标可行性：每月可存8000元，2年=19.2万，距离30万差10.8万。提供增额方案：1）每年涨薪按10%估算可多存约2-3万；2）年终奖/绩效按1-2个月工资计算约2-4万；3）用余钱做低风险理财（货币基金+债券基金组合约3%年化）；4）减少非必要支出（如奶茶外卖每月省500-1000元）。综合评估后给出差距分析和具体行动计划，推荐使用目标储蓄卡（如支付宝心愿储蓄或银行零存整取）强制储蓄。
